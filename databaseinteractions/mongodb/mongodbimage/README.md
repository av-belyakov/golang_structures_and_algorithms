# Настройка доступа к СУБД MongoDB

### MongoDB Compass

Скачать MongoDB Compass можно по ссылке https://downloads.mongodb.com/compass/mongodb-compass_1.49.2_amd6
заменив версию пакета на актуальную

### Автоматическое создание БД

БД с именем из переменной **MONGO_DATABASE** файла .env создаётся автоматически с помощью выполнения js скриптов находящихся в директории _init-scripts_. Все скрипты находящиеся в этой директории выполняются самой СУБД MongoDB **при первом** старте БД. Если БД была уже проинициализирована, то повторно эти скрипты **не выполняются**.

### Минималистичный web интерфейс Mongo Express

Веб-интерфейс Mongo Express, после запуска доступен на http://localhost:8081
